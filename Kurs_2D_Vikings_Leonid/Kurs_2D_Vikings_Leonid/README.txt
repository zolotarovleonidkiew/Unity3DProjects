
Управление:
=======================

[TAB] - переключение между викингами (если другие остались живи)
[E] - попытаться открыть дверь / использовать переключатель
[R] - использовать выбранный элемент в инвентаре
[Q] - переключиться на следующий элемент (выбрать) в инвентаре, даже если там пусто
[SPACE] - прыгает (высоко прыгает только красный, Ульрих)
[L-CONTROL] - Баеалог (зеленый) бьет мечом
[UP] - стоя на лифте, активироует лифт. Эта же нопка - ехать в обратную сторону.


Особенности
=======================

1. Олфа (со щитом) - блокирует все удары врагов, толкьо если они впереди викинга
2. Баелог - бьет мечом
3. Ульрих - прыгает
4. Можно подобрать, бросить и даже ПНУТЬ бомбу, чтобы она улетела. Убивает всех, в радиусе 3 клетки.
5. Разноцветные ключи - дял откртыия дверей, кусок мяса дает +1 к здоровью при максимуме 3 единицы.
6. В конце 2ого уровня виикнги улетает домой на корабле.
7. Уровень засчитывается, когда все 3 викинга пришил к табличке "EXIT"
